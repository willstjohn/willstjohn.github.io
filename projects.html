PacMen Exercise

GitHub Repository

This project was forked from an MIT exercise. In this project, the user is greeted with a blank page and two buttons: Add Pac Man and Start Game. The user can add as many Pac Men as they wish and when they hit Start Game they begin to bounce around the page, reflecting off the edges of the window, irrespective of window size.

Real Time Bus Tracking Exercise

GitHub Repository

This project was forked from an MIT exercise. It showcases a map of Boston and demonstrates several locations of stops for the MBTA bus. It includes some slight edits to the styling, resulting in a more muted and visually appealing look.

Eye Tracking Exercise

GitHub Repository
