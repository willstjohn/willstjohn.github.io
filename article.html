How to Make Your Own Personal Website

1. Create an account with GitHub

2. In an IDE of your choice, preferably Visual Studio Code, open or create a new folder with two files: index.html and a JPG image of yourself

3. Open index.html with your code editor and paste this code:

<html>
    <h1>Your Name’s Portfolio</h1>
    <img src="./IMG.jpg" />
</html>

Replace “Your Name's” with your name and replace IMG.jpg with the file name of your photo.

4. On your GitHub profile, go to Your Repositories and create a new repository.

5. Type yourusername.github.io in the Repository Name field, making sure to substitute your username in.

6. Hit the Create Repository button

7. After creating this repository, select Upload an Existing File and select the two files your just opened in Visual Studio Code

8. Hit the Commit Changes button

9. Wait a minute or two and in a web browser of your choice, navigate to your repository’s address i.e. yourusername.github.io

10. There you should see your photo and text that reads “Your Name’s Portfolio”

Congrats! You’ve just created your own personal website.
